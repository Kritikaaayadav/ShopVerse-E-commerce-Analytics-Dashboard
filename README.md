# ShopVerse E-commerce Analytics Dashboard

## Project Overview
ShopVerse is an interactive Power BI E-commerce Analytics Dashboard created to analyze sales performance, customer behavior, product trends, and marketing campaign effectiveness.

The project transforms raw business data into meaningful insights using Power BI, DAX, Power Query, and Excel datasets.

This dashboard helps businesses make data-driven decisions by tracking KPIs, customer engagement, revenue trends, product performance, and conversion behavior.

---

# Tools & Technologies Used
- Power BI
- DAX
- Power Query
- Microsoft Excel
- Data Modeling

- Interactive KPI dashboards

- Bookmark navigation

# Dashboard Pages

## 1. Overview Dashboard
Provides a summary of:
- Total Revenue
- Total Customers
- Total Orders
- Revenue Trends
- Top Products
- Revenue by Category & Brand

---

## 2. Sales Analysis Dashboard
Analyzes:
- Sales trends by hour
- Orders vs quantity
- Revenue by channel

- Category performance

---

## 3. Customers Analysis Dashboard
Analyzes:
- Customer acquisition channels
- Loyalty tier distribution
- Customer growth trends
- Customer conversion analysis
- Customer demographics

## 4. Products Analysis Dashboard
Analyzes:
- Product category performance
- Brand contribution
- Premium vs non-premium products

- Revenue trends by year

---

## 5. User Behavior & Campaign Insights Dashboard
Analyzes:
- Traffic source engagement
- User conversion funnel
- Device usage analysis

# Main DAX Measures Used

- Total Orders
- Total Customers
- Total Quantity

- Average Customer Spent

- Previous Orders
- Conversion Rate %
- Premium Products %

---

# Key Insights
- Electronics category generated the highest revenue.
- Organic and Paid Search channels brought maximum customers.
- Evening hours showed peak sales activity.
- Mobile users contributed the highest user sessions.
- Medium-price products generated the highest sales.
